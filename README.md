# School_District_Analysis

# Overview
In this repository, Thomas High School has been believed to have misleading reading and math scores for 9th graders due to academic dishonesty. Using Jupyter Notebook, Pandas, and Python, the scores will be replaced with NaNs before replicating a prior analysis to investigate the integrity of the grades. 

# Results

After replacing the ninth graders' math and reading scores with NaN scores, the following changes occurred for Thomas High School:

The overall passing percentage for Thomas High School fell to 65%
![image](https://user-images.githubusercontent.com/95376544/172530681-7fd87a30-c6ad-46a5-9958-c077efbcf258.png)


The overall passing percentage for the entire district fell to 64.9%
